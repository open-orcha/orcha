08 — Approvals (plan approval · task verification)

The two irreducibly human moments: approving an agent's plan before it burns turns, and verifying finished work before it counts as done ("never self-certify"). Both are modal sheets over the surfacing screen — the decision stays one thumb-reach away and the human never loses context. Spec: flows/08-approvals.md. Ships on the existing API (POST /api/decisions · POST /api/tasks/{tid}/verify).
