10 — Converse with an agent

Turn-based chat with one agent: your message wakes the agent, it works, it replies. Spec: flows/10-converse.md. All endpoints exist today; the composer is text-only in v1 (attachments = v2 even though the API already has them). The pending indicator is honest — pulsing dots only when the agent is actually working your turn; a "queued" notice when it's busy on a task. Frames mix platforms/themes; chat anatomy is pixel-equivalent.
